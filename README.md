# Asian Barometer Survey (ABS) Interactive Explorer

**Bridging Public Opinion Research and Data Visualization**
[Live Demo](https://abs-explorer-sigma.vercel.app)

The Asian Barometer Survey (ABS) is a regional network of research teams across Asia measuring public opinion on democracy, governance, and development. This interactive explorer provides a custom interface for navigating longitudinal data across multiple survey waves, making complex datasets accessible through dynamic visualization.

### 📊 Key Features
* **Wave-on-Wave Comparisons:** Track democratic trends and institutional trust across multiple survey cycles.
* **Country-Level Filtering:** Isolate specific political contexts for deeper comparative analysis.
* **Dynamic Visualization:** Custom Recharts implementation for longitudinal survey data.
* **Responsive Interface:** Built with Tailwind CSS for high-performance research workflows.

### 🛠 Technical Stack
* **Framework:** React + Vite
* **Styling:** Tailwind CSS
* **Charts:** Recharts
* **Deployment:** Vercel

### 🚀 Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
